Island
灵动岛
分类：Render
描述：显示一个酷炫的灵动岛。

需求
- 安全级别：常规模块
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
Island（灵动岛）用于显示一个酷炫的灵动岛。
适合在日常管理与自动化场景中按需启用。
初次使用可优先调整：Sync Notify Status、Sync Actionbar Status、Diversify Notify Icons。

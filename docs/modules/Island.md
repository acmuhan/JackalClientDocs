Island
灵动岛
分类：Render
描述：显示一个酷炫的灵动岛。

需求
- 安全级别：常规模块
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
Island 是 JackalClient 的 Render 分类模块。启用后会围绕 灵动岛 这一能力工作，核心行为是：显示一个酷炫的灵动岛。

# MouseESP（鼠标透视）

分类：Control
描述：高亮鼠标指针的位置，增强鼠标的视觉效果和可追踪性。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

MouseESP（鼠标透视）用于高亮鼠标指针的位置，增强鼠标的视觉效果和可追踪性。适合键鼠行为控制、输入增强和特殊视觉效果场景。

**初次使用建议**：优先调整 Visibility Check、Circle、Circle Radius 这三个核心配置。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "MouseESP" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 调整配置

**操作步骤**：
1. 右键点击 "MouseESP" 模块名称
2. 在弹出的配置窗口中调整所需选项
3. 调整完成后点击 "保存" 按钮

### 开始使用

**操作步骤**：
1. 启用模块后，鼠标指针会显示高亮效果
2. 移动鼠标，观察视觉效果
3. 根据需要调整不同的视觉元素配置
4. 当需要关闭鼠标透视效果时，再次点击模块开关按钮将其关闭

## 配置项

### 核心配置

- **Visibility Check（可见性检查）**
  类型：布尔；默认：true
  说明：控制是否启用鼠标可见性检查。默认值 true 代表启用可见性检查；若设置为 false，则禁用可见性检查。
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

- **Circle（圆圈）**
  类型：布尔；默认：false
  说明：控制是否显示鼠标周围的圆圈。默认值 false 代表不显示圆圈；若设置为 true，则显示圆圈。
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

- **Circle Radius（圆圈半径）**
  类型：数值；默认：25.0f
  说明：控制鼠标周围圆圈的半径。默认值 25.0f 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **Circle Opacity (0~1)（圆圈不透明度 (0~1)）**
  类型：数值；默认：0.3f
  说明：控制圆圈的不透明度。默认值 0.3f 兼顾可见性和遮挡；如果你觉得挡视线可小幅下调，若看不清可小幅上调。
  **操作指南**：在配置窗口中找到该选项，输入 0-1 之间的数值，点击保存

## 运行与测试

**测试步骤**：
1. 启用 MouseESP 模块
2. 调整不同的视觉元素配置
3. 移动鼠标，观察视觉效果
4. 测试鼠标点击时的效果
5. 在不同背景下测试视觉效果的可见性
6. 根据测试结果微调配置

**常见问题排查**：
- 如果鼠标透视效果不显示，检查模块是否已启用
- 如果视觉效果过于复杂，关闭一些不需要的元素
- 如果视觉效果在某些背景下不明显，调整颜色和不透明度

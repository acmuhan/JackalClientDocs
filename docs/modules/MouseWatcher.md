# MouseWatcher（鼠标监视）

分类：Control
描述：监视鼠标移动和交互，提供实时HUD信息。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

MouseWatcher（鼠标监视）用于监视鼠标移动和交互，提供实时HUD信息。适合键鼠行为控制、输入增强和自动化操作场景。

**初次使用建议**：优先调整 Movement Check Mode、Hide HUD When Menu On、HUD 这三个核心配置。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "MouseWatcher" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 调整配置

**操作步骤**：
1. 右键点击 "MouseWatcher" 模块名称
2. 在弹出的配置窗口中调整所需选项
3. 调整完成后点击 "保存" 按钮

### 开始使用

**操作步骤**：
1. 启用模块后，系统会自动开始监视鼠标移动和交互
2. 观察屏幕上显示的HUD信息
3. 根据需要调整运动检查模式和HUD显示方式
4. 模块会在后台持续运行，提供实时鼠标状态信息

## 配置项

### 核心配置

- **Movement Check（运动检查）**
  类型：枚举；默认："Ignore"
  说明：控制鼠标运动检查行为。默认值 Ignore 一般更稳妥；建议按使用场景逐个试用，而不是一次性切换多项。
  可选：Ignore（忽略）；Warn（警告）；Warn & Restore（警告并拉回）；Restore（还原）
  **操作指南**：在配置窗口中找到该选项，从下拉菜单中选择合适的模式

- **Movement Check Mode（运动检查模式）**
  类型：枚举；默认："Simple"
  说明：控制运动检查的严格程度。默认值 Simple 一般更稳妥；建议先保持默认，确认稳定后再切换比较效果。
  可选：Simple（简易）；Strict（严格）
  **操作指南**：在配置窗口中找到该选项，从下拉菜单中选择合适的模式

- **Hide HUD When Menu On（打开菜单时隐藏HUD）**
  类型：布尔；默认：true
  说明：控制菜单打开时是否隐藏HUD。默认值 true 一般更稳妥；建议先按默认值使用，确认行为符合预期后再调整。
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

### HUD 配置

- **HUD（是否显示HUD）**
  类型：枚举；默认："Fancy"
  说明：控制HUD显示方式。默认值 Fancy 一般更稳妥；建议按使用场景逐个试用，而不是一次性切换多项。
  可选：Off（关闭）；Simple（简易）；Fancy（优美）
  **操作指南**：在配置窗口中找到该选项，从下拉菜单中选择合适的显示方式

- **HUD Color（HUD 颜色）**
  类型：枚举；默认："White"
  说明：控制HUD显示颜色。建议先选对比度高的配色保证可读性；若是动态颜色，注意在复杂背景下的辨识度。
  可选：见 [NAMED_COLOR_BASE_LIST](./NAMED_COLOR_BASE_LIST.md)
  **操作指南**：在配置窗口中找到该选项，从下拉菜单中选择合适的颜色

- **HUD Font Size（HUD 字号）**
  类型：数值；默认：20
  说明：控制HUD文本大小。默认字号 20 适合多数分辨率；高分屏可适当加大，低分辨率建议减少以免拥挤。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **HUD Line Gap（HUD 行距）**
  类型：数值；默认：5
  说明：控制HUD文本行间距。默认值 5 通常是平衡视觉效果与紧凑度的设置，建议小步调整并实时观察。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **HUD X Rate（HUD 横坐标比例）**
  类型：数值；默认：0.01f
  说明：控制HUD在屏幕中的水平位置。默认值 0.01f 一般是作者调过的稳定布局；建议每次只改一个轴，避免元素跑出可视区域。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **HUD Y Rate（HUD 纵坐标比例）**
  类型：数值；默认：0.44f
  说明：控制HUD在屏幕中的垂直位置。默认值 0.44f 一般是作者调过的稳定布局；建议每次只改一个轴，避免元素跑出可视区域。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

### 高级配置

- **Sample Cooldown（采样冷却）**
  类型：数值；默认：10U
  说明：控制检测/刷新/动画节奏。默认值 10U 以稳定为主；调小会更灵敏但可能增加资源占用，调大则更省资源但响应更慢。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **Acceleration Threshold（加速度阈值）**
  类型：数值；默认：800.0f
  说明：控制鼠标加速度检测的灵敏度。默认值 800.0f 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **Strict Check Threshold（严格检查阈值）**
  类型：数值；默认：10
  说明：控制严格检查模式下的阈值。默认值 10 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

## 运行与测试

**测试步骤**：
1. 启用 MouseWatcher 模块
2. 调整HUD显示方式和位置
3. 移动鼠标，观察HUD信息的实时更新
4. 测试不同的运动检查模式
5. 在不同背景下测试HUD的可见性
6. 根据测试结果微调配置

**常见问题排查**：
- 如果HUD不显示，检查模块是否已启用
- 如果HUD位置不合适，调整 HUD X Rate 和 HUD Y Rate 值
- 如果运动检查过于敏感，调整 Acceleration Threshold 值

# MouseSwap（鼠标交换）

分类：Control
描述：交换鼠标左右键功能。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

MouseSwap（鼠标交换）用于交换鼠标左右键功能。适合键鼠行为控制、输入增强和特殊使用场景。

**初次使用建议**：启用模块后，直接测试鼠标左右键功能是否已交换。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "MouseSwap" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 开始使用

**操作步骤**：
1. 启用模块后，系统会自动交换鼠标左右键功能
2. 测试鼠标左键和右键的功能是否已交换
3. 当需要恢复默认鼠标行为时，再次点击模块开关按钮将其关闭

## 配置项

- **无独立配置项**：该模块在 default_settings.h 中未定义独立配置项，启用即可使用

## 运行与测试

**测试步骤**：
1. 启用 MouseSwap 模块
2. 测试鼠标左键和右键的功能是否已交换
3. 尝试使用鼠标进行日常操作，确认交换效果
4. 关闭模块，确认鼠标行为恢复正常
5. 根据测试结果确认模块是否正常工作

**常见问题排查**：
- 如果鼠标键功能未交换，检查模块是否已启用
- 如果交换后操作不习惯，可随时关闭模块恢复默认行为

# AntiAFK（反挂机）

分类：Control
描述：自动检测并防止系统进入挂机状态。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

AntiAFK（反挂机）用于自动检测并防止系统进入挂机状态。适合键鼠行为控制、输入增强和自动化操作场景。

**初次使用建议**：优先调整 Move Cooldown、Move Duration 这两个核心配置。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "AntiAFK" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 调整配置

**操作步骤**：
1. 右键点击 "AntiAFK" 模块名称
2. 在弹出的配置窗口中调整所需选项
3. 调整完成后点击 "保存" 按钮

### 开始使用

**操作步骤**：
1. 启用模块后，系统会自动检测用户活动状态
2. 当检测到长时间无活动时，会自动执行轻微的鼠标移动
3. 观察反挂机效果，根据需要调整移动冷却和时长
4. 模块会在后台持续运行，无需手动干预

## 配置项

### 核心配置

- **Move Cooldown（移动冷却）**
  类型：数值；默认：2000L
  说明：控制检测/刷新/动画节奏。默认值 2000L 以稳定为主；调小会更灵敏但可能增加资源占用，调大则更省资源但响应更慢。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **Move Duration（移动时长）**
  类型：数值；默认：3000L
  说明：控制检测/刷新/动画节奏。默认值 3000L 以稳定为主；调小会更灵敏但可能增加资源占用，调大则更省资源但响应更慢。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

## 运行与测试

**测试步骤**：
1. 启用 AntiAFK 模块
2. 调整移动冷却和时长值
3. 保持计算机静止，观察模块是否自动执行移动
4. 测试不同时间长度的无活动状态
5. 根据测试结果微调配置

**常见问题排查**：
- 如果反挂机功能不生效，检查模块是否已启用
- 如果移动过于频繁，增加 Move Cooldown 值
- 如果移动不够及时，减小 Move Cooldown 值

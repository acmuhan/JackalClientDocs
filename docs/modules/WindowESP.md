介绍
WindowESP 是 JackalClient 的 Window 分类模块。启用后会围绕 窗口透视 这一能力工作，核心行为是：暂无描述

配置项
- Box Color：类型=枚举，默认="Rainbow"
- Box Alpha：类型=数值，默认=0.5f
- Hide HUD When Menu On：类型=布尔，默认=true
- Exclude Foreground：类型=布尔，默认=true
- Exclude Pointed：类型=布尔，默认=true
- Exclude Invisible：类型=布尔，默认=true
- Box Line Width：类型=数值，默认=3.0f

历史更新

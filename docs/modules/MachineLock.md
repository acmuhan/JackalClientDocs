MachineLock
锁机
分类：Combat
描述：锁定这台计算机（谨慎使用）。

需求
- 安全级别：恶意模块（高风险）
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍

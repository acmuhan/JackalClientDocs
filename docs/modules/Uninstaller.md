Uninstaller
一键卸载
分类：未分类
描述：自动帮你卸载一些软件。

需求
- 安全级别：恶意模块（高风险）
- 权限需求：管理员
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
Uninstaller 是 JackalClient 的 未分类 分类模块。启用后会围绕 一键卸载 这一能力工作，核心行为是：自动帮你卸载一些软件。

配置项
- Transparent Window：类型=布尔，默认=false
- Uninstall Start Method：类型=枚举，默认="Injection"
- Mode：类型=枚举，默认="Auto"
- Notify Steps：类型=布尔，默认=true
- Victim：类型=枚举，默认="None"

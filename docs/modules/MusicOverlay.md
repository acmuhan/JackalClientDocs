音乐信息
[PRO]
分类：Render
描述：显示一个覆盖层，展示音乐信息。（专业版）

需求
- 安全级别：常规模块
- 权限需求：无
- 驱动依赖：否

- 版本属性：PRO 独有

介绍
MusicOverlay（音乐信息）用于显示一个覆盖层，展示音乐信息。（专业版）。
适合在日常管理与自动化场景中按需启用。
初次使用可优先调整：Detect Mode、Temp HUD Duration (ms)、Progress Bar Update Cooldown (ms)。

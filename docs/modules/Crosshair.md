# Crosshair（准星线）

分类：Control
描述：在屏幕中心显示准星线，提高瞄准精度。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

Crosshair（准星线）用于在屏幕中心显示准星线，提高瞄准精度。适合键鼠行为控制、输入增强和游戏辅助场景。

**初次使用建议**：优先调整 Advanced Mode、Advanced Mode Show HUD、Advanced Mode Hotkey 这三个核心配置。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "Crosshair" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 调整配置

**操作步骤**：
1. 右键点击 "Crosshair" 模块名称
2. 在弹出的配置窗口中调整所需选项
3. 调整完成后点击 "保存" 按钮

### 切换高级模式

**操作步骤**：
1. 启用 Crosshair 模块
2. 按下默认快捷键：左右 Alt 键同时按下
3. 观察屏幕上是否显示准星线和 HUD 信息
4. 再次按下相同快捷键可关闭高级模式

## 配置项

### 基本配置

- **Crosshair Line Width**（准星线宽）
  类型：数值；默认：2
  说明：调整准星线的粗细，建议根据屏幕分辨率和个人喜好调整
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **Crosshair Line Opacity (0~1)**（准星线不透明度）
  类型：数值；默认：0.9f
  说明：调整准星线的透明程度，兼顾可见性和遮挡
  **操作指南**：在配置窗口中找到该选项，输入 0-1 之间的数值，点击保存

- **Crosshair Line Color**（准星线颜色）
  类型：枚举；默认："Rainbow"
  说明：选择准星线的颜色
  可选：White（白色）；Red（红色）；Green（绿色）；Blue（蓝色）；Magenta（品红色）；Colorful（缤纷）；Rainbow（彩虹色）
  **操作指南**：在配置窗口中找到该选项，从下拉菜单中选择喜欢的颜色

- **Crosshair Line Combination Type**（准星线组合类型）
  类型：枚举；默认："Plus"
  说明：选择准星线的样式
  可选：Horizontal（横向）；Vertical（纵向）；Plus（加号）；X（未收录）；Plus & X（米字型）
  **操作指南**：在配置窗口中找到该选项，从下拉菜单中选择喜欢的样式

### 高级配置

- **Advanced Mode**（高级模式）
  类型：布尔；默认：false
  说明：启用高级模式，显示更多信息和功能
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

- **Advanced Mode Show HUD**（高级模式显示文本）
  类型：布尔；默认：true
  说明：在高级模式下显示 HUD 信息
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

- **Advanced Mode Hotkey**（高级模式快捷键）
  类型：按键/复合；默认：左右 Alt 键
  **操作指南**：在配置窗口中找到该选项，点击后按下想要设置的按键组合

- **Crosshair Hollow Enabled**（鼠标周围不绘制准星线）
  类型：布尔；默认：true
  说明：在鼠标周围区域不绘制准星线，避免遮挡鼠标指针
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

- **Crosshair Hollow Radius**（鼠标周围准星不绘制的半径）
  类型：数值；默认：80
  说明：调整鼠标周围不绘制准星线的区域大小
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

### 显示配置

- **Coord Font Size**（坐标字号）
  类型：数值；默认：30
  说明：调整高级模式下坐标显示的字体大小
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **HUD Font Size**（HUD 字号）
  类型：数值；默认：25
  说明：调整高级模式下 HUD 信息的字体大小
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **HUD Line Gap**（HUD 行距）
  类型：数值；默认：5
  说明：调整 HUD 信息的行间距
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

- **HUD Opacity (0~1)**（HUD 不透明度）
  类型：数值；默认：0.9f
  说明：调整 HUD 信息的透明程度
  **操作指南**：在配置窗口中找到该选项，输入 0-1 之间的数值，点击保存

## 运行与测试

**测试步骤**：
1. 启用 Crosshair 模块
2. 调整准星线的样式、颜色和大小
3. 移动鼠标，观察准星线是否跟随
4. 切换高级模式，测试 HUD 信息显示
5. 在不同背景下测试准星线的可见性
6. 根据测试结果微调配置

**常见问题排查**：
- 如果准星线不显示，检查模块是否已启用
- 如果准星线遮挡鼠标，调整 Crosshair Hollow Radius 值
- 如果准星线在某些背景下不明显，尝试更换颜色或增加不透明度

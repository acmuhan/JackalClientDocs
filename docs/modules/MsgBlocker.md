配置项
- Hide HUD When Menu On：类型=布尔，默认=false
- Ignore WM_QUIT：类型=布尔，默认=false
- Ignore WM_TIMER：类型=布尔，默认=false
- Output Messages：类型=布尔，默认=true
- Notify：类型=布尔，默认=true
- Overlay Opacity (0~1)：类型=数值，默认=0.2f
- Disable When Quit：类型=布尔，默认=true
- Ignore WM_PAINT：类型=布尔，默认=true
- Overlay：类型=布尔，默认=true
- Overlay Color：类型=枚举，默认="Red"
- Target Window：类型=枚举，默认="Foreground"

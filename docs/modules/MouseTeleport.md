# MouseTeleport（鼠标传送）

分类：Control
描述：在某些场合下传送鼠标，实现鼠标在屏幕边缘的无缝穿越。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

MouseTeleport（鼠标传送）用于在某些场合下传送鼠标，实现鼠标在屏幕边缘的无缝穿越。适合键鼠行为控制、输入增强和自动化操作场景。

**初次使用建议**：优先调整 Edge Teleport、Teleport Sound 这两个核心配置。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "MouseTeleport" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 调整配置

**操作步骤**：
1. 右键点击 "MouseTeleport" 模块名称
2. 在弹出的配置窗口中调整所需选项
3. 调整完成后点击 "保存" 按钮

### 开始使用

**操作步骤**：
1. 启用模块后，将鼠标移动到屏幕边缘
2. 当鼠标到达屏幕边缘时，会自动传送到对面边缘
3. 观察传送效果，根据需要调整配置
4. 当需要关闭鼠标传送功能时，再次点击模块开关按钮将其关闭

## 配置项

### 核心配置

- **Edge Teleport（边缘传送）**
  类型：布尔；默认：true
  说明：控制是否启用鼠标边缘传送功能。默认值 true 代表启用边缘传送；若设置为 false，则禁用边缘传送。
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

- **Teleport Sound（传送音效）**
  类型：布尔；默认：true
  说明：控制是否启用传送音效。默认值 true 代表启用传送音效；若设置为 false，则禁用传送音效。
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

## 运行与测试

**测试步骤**：
1. 启用 MouseTeleport 模块
2. 调整 Edge Teleport 和 Teleport Sound 选项
3. 将鼠标移动到屏幕边缘，测试是否能自动传送到对面边缘
4. 测试不同屏幕边缘的传送效果
5. 观察传送时是否有音效反馈
6. 根据测试结果微调配置

**常见问题排查**：
- 如果鼠标边缘传送不生效，检查模块是否已启用
- 如果传送时没有音效，检查 Teleport Sound 选项是否已开启
- 如果传送位置不准确，可能需要调整屏幕分辨率和缩放设置

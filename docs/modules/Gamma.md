配置项
- Alpha（不透明度）
 类型：数值；默认：255
 说明：这是数值型配置。默认值 255 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
- Set Once（单次设置时间点）
 类型：布尔；默认：false
 说明：这是开关型配置。默认值 false 代表作者推荐的初始行为；若要改动，建议一次只改一个开关便于观察影响。
- Disable When Quit（退出时关闭）
 类型：布尔；默认：true
 说明：这是行为开关项。建议先按默认值使用，确认行为符合预期后再逐项启停，避免多个开关同时改动造成排查困难。

# MouseDisabler（禁用鼠标）

分类：Control
描述：临时禁用鼠标功能，防止误操作。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

MouseDisabler（禁用鼠标）用于临时禁用鼠标功能，防止误操作。适合键鼠行为控制、输入增强和自动化操作场景。

**初次使用建议**：优先调整 Allow Move、ALLOW NO KEYBIND 这两个核心配置。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "MouseDisabler" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 调整配置

**操作步骤**：
1. 右键点击 "MouseDisabler" 模块名称
2. 在弹出的配置窗口中调整所需选项
3. 调整完成后点击 "保存" 按钮

### 开始使用

**操作步骤**：
1. 启用模块后，系统会根据配置禁用鼠标功能
2. 如果需要临时启用鼠标移动，确保 Allow Move 选项已开启
3. 当需要完全恢复鼠标功能时，再次点击模块开关按钮将其关闭

## 配置项

### 核心配置

- **Allow Move（允许移动）**
  类型：布尔；默认：false
  说明：控制是否允许鼠标移动。默认值 false 代表完全禁用鼠标；若设置为 true，则只禁用鼠标点击，允许鼠标移动。
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

- **ALLOW NO KEYBIND（允许不绑定按键）**
  类型：布尔；默认：false
  说明：控制是否允许不绑定按键。默认值 false 代表必须绑定按键才能启用模块；若设置为 true，则可以不绑定按键直接启用。
  **操作指南**：在配置窗口中找到该选项，点击开关按钮

## 运行与测试

**测试步骤**：
1. 启用 MouseDisabler 模块
2. 测试鼠标是否已被禁用
3. 如果 Allow Move 为 true，测试鼠标是否可以移动但不能点击
4. 当需要恢复鼠标功能时，关闭模块
5. 测试鼠标功能是否已完全恢复
6. 根据测试结果微调配置

**常见问题排查**：
- 如果鼠标未被禁用，检查模块是否已启用
- 如果无法启用模块，检查是否已绑定按键或 ALLOW NO KEYBIND 是否为 true
- 如果需要临时启用鼠标，确保 Allow Move 选项已开启

BiliConsole
B站控制台
分类：Web
描述：一个实用的交互式B站控制台。

需求
- 安全级别：常规模块
- 权限需求：无
- 驱动依赖：否
- 联网需求：是
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
BiliConsole（B站控制台）用于一个实用的交互式B站控制台。
适合联网信息获取、账号联动和在线交互场景。
初次使用可优先调整：Draw Cooldown (ms)、Update Cooldown (ms)。
使用前请确认网络可用及相关 API/平台账号配置完整。

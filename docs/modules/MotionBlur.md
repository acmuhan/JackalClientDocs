MotionBlur
动态模糊
分类：Render
描述：在屏幕上应用动态模糊。

需求
- 安全级别：常规模块
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
MotionBlur（动态模糊）用于在屏幕上应用动态模糊。
适合在日常管理与自动化场景中按需启用。
初次使用可优先调整：Apply Model、Update Cooldown。

配置项
- Apply Area（应用区域）
 类型：枚举；默认："Screen"
 说明：这是选项型配置。默认值 Screen 一般更稳妥；建议按使用场景逐个试用，而不是一次性切换多项。
 可选：Client Only（仅客户端）；Screen（屏幕）
- Apply Model（应用模型）
 类型：枚举；默认："Exponential"
 说明：这是该模块的核心行为开关，不同选项对应不同执行策略。建议先保持默认 Exponential ，确认稳定后再逐项切换比较效果。
 可选：Gaussian（高斯）；Exponential（指数）
- Update Cooldown（更新冷却）
 类型：数值；默认：40
 说明：用于控制检测/刷新/动画节奏。默认值 40 以稳定为主；调小会更灵敏但可能增加资源占用，调大则更省资源但响应更慢。

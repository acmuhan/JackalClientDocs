配置项
- Max History Length：类型=数值，默认=500
- Command Hints：类型=布尔，默认=true
- Title Color：类型=枚举，默认="Rainbow"
- Batch File Error Level：类型=枚举，默认="Only Exception"
- Close Menu：类型=布尔，默认=true
- Custom Input Prompt：类型=文本，默认=">>>"
- Batch File Execution Notify：类型=布尔，默认=true
- Command Completion：类型=布尔，默认=true
- Echo Command：类型=布尔，默认=false
- Input Color：类型=文本，默认="white"
- Using Environment Variables：类型=布尔，默认=true
- bind：类型=按键/复合，默认={{"Keybind", {VK_RMENU, 'S'}}}
- Command Highlighter：类型=布尔，默认=true
- Chatter Color：类型=枚举，默认="White"
- Command Max Length：类型=数值，默认=65535L
- Theme：类型=枚举，默认="Jackal Pro"
- Using Builtin Variables：类型=布尔，默认=true
- Jackal Pro Subtheme：类型=枚举，默认="Simple"
- Allow Terminate Shell Thread：类型=布尔，默认=false
- Exclusive Mode：类型=布尔，默认=false
- Input Prompt：类型=枚举，默认="$>"

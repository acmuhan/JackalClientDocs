Arraylist
模块列表
分类：Render
描述：显示已启用模块列表。

需求
- 安全级别：常规模块
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
Arraylist（模块列表）用于显示已启用模块列表。
适合在日常管理与自动化场景中按需启用。
初次使用可优先调整：Background Height Mode。

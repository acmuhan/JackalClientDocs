介绍
Translator 是 JackalClient 的 未分类 分类模块。启用后会围绕 翻译器 这一能力工作，核心行为是：一个简单的翻译服务。

配置项
- Module Status Auto Switch：类型=布尔，默认=true
- Mode：类型=枚举，默认="PearAPI"
- AI Prompt：类型=文本，默认="将冒号后面的文本翻译成{lang}，直接输出即可，不要输出其他任何内容：\n{text}"

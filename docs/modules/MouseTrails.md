# MouseTrails（鼠标轨迹）

分类：Control
描述：为鼠标添加轨迹效果，增强鼠标移动的视觉反馈。

## 需求

- **安全级别**：常规模块
- **权限需求**：无
- **驱动依赖**：否
- **联网需求**：否
- **开发状态**：稳定/常规
- **版本属性**：普通可用

## 介绍

MouseTrails（鼠标轨迹）用于为鼠标添加轨迹效果，增强鼠标移动的视觉反馈。适合键鼠行为控制、输入增强和特殊视觉效果场景。

**初次使用建议**：优先调整 Trails 配置，控制轨迹长度。

## 核心操作

### 启用模块

**操作步骤**：
1. 在左侧菜单栏中找到 "Control" 分类
2. 点击展开分类列表
3. 找到 "MouseTrails" 模块
4. 点击模块名称左侧的开关按钮，使其变亮

### 调整配置

**操作步骤**：
1. 右键点击 "MouseTrails" 模块名称
2. 在弹出的配置窗口中调整所需选项
3. 调整完成后点击 "保存" 按钮

### 开始使用

**操作步骤**：
1. 启用模块后，移动鼠标时会显示轨迹效果
2. 观察鼠标轨迹效果，根据需要调整轨迹长度
3. 当需要关闭鼠标轨迹效果时，再次点击模块开关按钮将其关闭

## 配置项

### 核心配置

- **Trails（轨迹）**
  类型：数值；默认：5
  说明：控制鼠标轨迹的长度。默认值 5 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
  **操作指南**：在配置窗口中找到该选项，输入合适的数值，点击保存

## 运行与测试

**测试步骤**：
1. 启用 MouseTrails 模块
2. 调整 Trails 值，设置不同的轨迹长度
3. 移动鼠标，观察轨迹效果
4. 在不同背景下测试轨迹的可见性
5. 测试鼠标移动速度对轨迹效果的影响
6. 根据测试结果微调配置

**常见问题排查**：
- 如果鼠标轨迹不显示，检查模块是否已启用
- 如果轨迹效果不明显，增加 Trails 值
- 如果轨迹效果过于明显，减少 Trails 值

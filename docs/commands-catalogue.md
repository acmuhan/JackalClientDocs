## 普通 / PRO 分级速查

| 章节 | 主要用途 | 普通版可用性 | PRO扩展密度 |
| :-- | :-- | :-- | :-- |
| 系统命令 | 进程/窗口/权限/系统状态 | 中 | 高 |
| 文件命令 | 文件与目录操作 | 高 | 低 |
| 网络命令 | 网络状态、连接与代理 | 中 | 中 |
| B站命令 | 内容与账号流程 | 中-高 | 中 |
| 文本处理命令 | 文本转换与批处理 | 高 | 低 |

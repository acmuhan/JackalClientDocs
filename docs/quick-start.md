## 1. 启动方式

常见启动文件：

- `Launcher.exe`：启动器（可检查更新）
- `Release/JackalClient.exe`：客户端主程序

## 2. GUI 基本操作

默认热键：

- 右 `Ctrl` / 右 `Shift`：打开或关闭菜单（ClickGUI）

模块操作：

- 左键：开关模块
- 右键：进入模块配置

辅助操作：

- `Ctrl + F`：打开模块搜索
- 方向键/滚轮：移动和滚动内容

## 3. Shell/命令行基本操作

命令行示例：

- `/help`：查看帮助
- `/list`：列出模块
- `/enable <模块名>`：启用模块
- `/disable <模块名>`：禁用模块
- `/quit`：退出

## 4. 建议先做的配置

1. 设置 GUI 热键，避免和系统热键冲突
2. 检查高风险模块开关（建议默认关闭）
3. 如果需要命令操作，再启用 Shell
4. 首次使用先做一次配置备份

## 5. 常见问题入口

- 常见问题：`/faq`
- BetterLyrics 专题：`/betterlyrics`

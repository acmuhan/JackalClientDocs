## 准备阶段

### 第一步：找到启动文件

进入 JackalClient 安装目录，你会看到两个主要启动文件：

- `Launcher.exe`：启动器（可检查更新）
- `Release/JackalClient.exe`：客户端主程序

双击 `Launcher.exe` 启动程序，等待几秒钟后客户端界面会出现。

### 第二步：认识界面布局

程序启动后，你会看到以下主要区域：

- **左侧菜单栏**：包含所有功能模块的分类导航
- **中央显示区**：显示当前选中模块的详细信息和配置项
- **底部状态栏**：显示程序运行状态和提示信息
- **右上角图标**：用于打开设置菜单和帮助文档

现在，你已经了解了 JackalClient 的基本结构，接下来开始核心操作。

## 核心操作

### 操作一：打开图形界面

按下键盘右侧的 `Ctrl` 键或 `Shift` 键，图形界面会立即弹出。

再次按下相同的键，界面会关闭。

**提示**：如果这个热键与你的其他程序冲突，可以在设置中修改为其他组合键。

### 操作二：开关模块

在图形界面中：

1. 在左侧菜单栏中找到你想要的模块分类
2. 点击分类名称展开模块列表
3. 找到目标模块，点击模块名称左侧的开关按钮
4. 按钮变亮表示模块已启用，变暗表示已禁用

**示例**：在"Control"分类下找到"AutoClicker"，点击开关即可启用连点器功能。

### 操作三：调整模块配置

1. 在图形界面中右键点击任意模块名称
2. 弹出的配置窗口会显示该模块的所有可调选项
3. 点击选项右侧的数值或下拉框进行修改
4. 修改完成后点击"保存"按钮

**示例**：右键点击"AutoClicker"模块，将"Left CPS Max"从默认的 14 调整为 20，保存后生效。

### 操作四：使用命令行

按下 `~` 键（波浪号）或 `/` 键打开命令行输入框。

输入以下命令并按回车：

```text
/help
```

屏幕上会显示所有可用命令的帮助信息。

输入以下命令查看模块列表：

```text
/list
```

输入以下命令启用模块：

```text
/enable AutoClicker
```

输入以下命令禁用模块：

```text
/disable AutoClicker
```

### 操作五：快速搜索模块

在图形界面中按下 `Ctrl + F` 组合键，顶部会出现搜索框。

输入模块名称的关键词（如"Clicker"），界面会自动过滤显示匹配的模块。

使用方向键上下移动，按回车键选中目标模块。

## 运行与测试

### 测试一：验证模块功能

1. 启用"AutoClicker"模块
2. 将鼠标移动到任意可点击的区域
3. 按下鼠标侧键（默认为 XButton2），连点器会开始工作
4. 观察鼠标点击频率是否符合你设置的 CPS 值
5. 再次按下侧键，连点器停止工作

### 测试二：检查命令执行

在命令行中输入以下命令：

```text
/ver
```

查看底部状态栏显示的版本信息，确认命令已成功执行。

输入以下命令：

```text
/notify 测试通知
```

屏幕右下角会弹出通知提示框，验证通知功能正常。

### 测试三：验证配置保存

1. 修改任意模块的配置项
2. 在命令行中输入：
   ```text
   /save
   ```
3. 重启客户端
4. 检查你修改的配置是否依然生效

## 进阶建议

### 建议一：创建快捷键

1. 在图形界面中找到"Keybind"配置项，点击右侧的输入框
2. 按下你想要设置的快捷键组合（如 `Ctrl + Alt + A`）
3. 保存配置后，按下这个组合键即可快速启用该模块

### 建议二：配置多个档案

在命令行中输入：

```text
/profile save 游戏模式
```

将当前所有模块配置保存为"游戏模式"档案。

输入以下命令创建另一个档案：

```text
/profile save 工作模式
```

在不同场景下切换档案：

```text
/profile load 游戏模式
```

### 建议三：组合使用模块

尝试同时启用多个模块，实现更复杂的功能组合。

**示例组合**：
- 启用"AutoClicker"实现自动点击
- 启用"Keystrokes"显示按键状态
- 启用"Crosshair"显示准星线

这样的组合可以在游戏中提供更完整的辅助体验。

### 建议四：使用命令别名

在命令行中输入：

```text
/helper add ac AutoClicker
```

创建别名"ac"指向"AutoClicker"模块。

以后可以直接输入：

```text
/enable ac
```

快速启用该模块，节省输入时间。

## 常见问题入口

- 常见问题：在命令行输入 `/faq`
- BetterLyrics 专题：在命令行输入 `/betterlyrics`

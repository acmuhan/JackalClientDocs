# 命令系统总览

## 命令手册导航

**命令总览页已升级为多页手册**，提供更详细的命令说明和使用示例。

**访问入口：** [`/commands/`](/commands/)

## 命令分类

- **系统命令**：系统管理、进程操作、服务控制等
- **文件命令**：文件操作、目录管理、内容处理等
- **网络命令**：网络配置、连接管理、信息查询等
- **B站命令**：哔哩哔哩专属功能、视频操作、弹幕管理等
- **文本处理命令**：文本分析、转换、生成等

## 命令更新

**当命令有更新时，执行以下操作：**

1. 打开命令行终端
2. 进入 JackalClientDocs 目录
3. 执行命令：
   ```bash
   npm run docs:commands
   ```
4. 等待文档生成完成

## 开始使用命令

**操作步骤：**
1. 打开 JackalClient
2. 找到并启用 "命令行" 模块 (Shell)
3. 在弹出的控制台中输入命令
4. 按 Tab 键使用命令补全功能
5. 输入 `/help` 查看命令帮助信息

**示例**：输入 `/help system` 查看系统相关命令的详细说明

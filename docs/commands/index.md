## 开始使用命令

**操作步骤：**
1. 打开 JackalClient
2. 找到并启用 "命令行" 模块 (Shell)
3. 在弹出的控制台中输入命令
4. 按 Tab 键使用命令补全功能
5. 输入 `/help` 查看命令帮助信息

**示例**：输入 `/help system` 查看系统相关命令的详细说明

## 命令更新

**当命令有更新时，执行以下操作：**

1. 打开命令行终端
2. 进入 JackalClientDocs 目录
3. 执行命令：
   ```bash
   npm run docs:commands
   ```
4. 等待文档生成完成

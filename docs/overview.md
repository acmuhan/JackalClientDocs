# JackalClient 文档概览

## 什么是 JackalClient？

JackalClient 是一个面向 Windows 的客户端工具箱，定位为学习与研究用途的软件集合。它不是一个单一功能程序，而是由多个可独立开关的模块组成的完整平台。

## 核心模块组成

- **图形界面模块**（ClickGUI / HUD）：提供直观的可视化操作界面
- **命令行与 Shell 模块**：包含大量命令，支持快速调用和组合功能
- **系统工具模块**：涵盖窗口、进程、音量、时间、文件等系统操作
- **自动化模块**：支持启动、任务、规则触发等自动化操作
- **在线能力模块**：集成 B 站、网易云、文本数据库等在线服务

## 项目定位

- **平台**：Windows（64 位为主）
- **交互**：GUI + 命令行双栈操作
- **目标**：效率辅助、系统操作、自动化和信息处理
- **许可与风险**：仅限合法、合规、可控场景使用

## 推荐阅读路径

1. **第一步**：阅读《快速开始》，了解如何启动程序、开关 GUI、操作模块和退出
2. **第二步**：浏览《功能与模块地图》，明确 JackalClient 提供的所有能力
3. **第三步**：查阅《完整命令手册》，掌握命令行操作的全部功能
4. **第四步**：学习《安全与使用边界》，理解使用过程中的风险与约束

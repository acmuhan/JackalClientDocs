## 模块分级速查（普通 / PRO）

| 场景 | 代表能力 | 普通版覆盖 | PRO增强 |
| :-- | :-- | :-- | :-- |
| 界面与显示 | HUD、动画、样式、可视化 | 高 | 中-高 |
| 系统与进程工具 | 进程/窗口/服务控制 | 中 | 高 |
| 自动化与编排 | 触发器、脚本、宏 | 中 | 高 |
| 在线能力 | B站、歌词、在线检索 | 中 | 中-高 |

## A. 界面与显示

- ClickGUI / HUD 编辑
- ArrayList / Island / ChatBox 等 UI 组件
- 多种动画、阴影、样式与字体配置

适用场景：

- 个性化面板
- 运行状态可视化
- 通知与信息聚合

## B. 系统与进程工具

- 进程、窗口、服务枚举与控制
- 音量管理、设备管理、时间管理
- 文件监视、文件管理与恢复

适用场景：

- 本机运维与排查
- 系统行为自动化
- 工具化运作

## C. 自动化与操作编排

- 模块联动
- 快捷键触发
- 命令脚本（`.jbat` / 宏）
- 定时、条件触发

适用场景：

- 重复操作自动化
- 高效流程串联

## D. 在线能力

- Bilibili：用户、视频、评论、直播相关命令
- 网易云：歌曲、歌词、覆盖层/歌词增强相关流程
- 文本数据库与在线检索

适用场景：

- 内容创作辅助
- 在线信息聚合

## E. 专业版（PRO）能力

历史记录显示存在大量 PRO 专属命令与模块扩展，例如：

- 更完整的在线接口能力
- 扩展 GUI 与动画能力
- 更复杂的自动化/筛选逻辑

## 使用指南

**操作步骤**：
1. **第一步**：在左侧菜单栏中找到模块分类
2. **第二步**：点击分类名称展开模块列表
3. **第三步**：找到目标模块，点击开关按钮启用
4. **第四步**：右键点击模块名称调整配置
5. **第五步**：根据需要使用相关命令

## A. 界面与显示

### 包含模块

- ClickGUI / HUD 编辑
- ArrayList / Island / ChatBox 等 UI 组件
- 多种动画、阴影、样式与字体配置

### 适用场景

- 个性化面板
- 运行状态可视化
- 通知与信息聚合

### 操作指南

**启用界面模块**：
1. 在左侧菜单栏中找到 "界面与显示" 分类
2. 点击展开分类列表
3. 找到目标模块，点击开关按钮启用
4. 右键点击模块名称调整外观配置

## B. 系统与进程工具

### 包含模块

- 进程、窗口、服务枚举与控制
- 音量管理、设备管理、时间管理
- 文件监视、文件管理与恢复

### 适用场景

- 本机运维与排查
- 系统行为自动化
- 工具化运作

### 操作指南

**使用系统工具**：
1. 在左侧菜单栏中找到 "系统与进程工具" 分类
2. 点击展开分类列表
3. 找到目标模块，点击开关按钮启用
4. 根据模块功能执行相应操作

## C. 自动化与操作编排

### 包含模块

- 模块联动
- 快捷键触发
- 命令脚本（`.jbat` / 宏）
- 定时、条件触发

### 适用场景

- 重复操作自动化
- 高效流程串联

### 操作指南

**设置自动化**：
1. 在左侧菜单栏中找到 "自动化与操作编排" 分类
2. 点击展开分类列表
3. 找到目标模块，点击开关按钮启用
4. 右键点击模块名称配置触发条件
5. 测试自动化流程是否正常执行

## D. 在线能力

### 包含模块

- Bilibili：用户、视频、评论、直播相关命令
- 网易云：歌曲、歌词、覆盖层/歌词增强相关流程
- 文本数据库与在线检索

### 适用场景

- 内容创作辅助
- 在线信息聚合

### 操作指南

**使用在线能力**：
1. 在左侧菜单栏中找到 "在线能力" 分类
2. 点击展开分类列表
3. 找到目标模块，点击开关按钮启用
4. 根据需要登录相关服务账号
5. 使用相关命令执行在线操作

## E. 专业版（PRO）能力

### 专属功能

历史记录显示存在大量 PRO 专属命令与模块扩展，例如：

- 更复杂的自动化/筛选逻辑

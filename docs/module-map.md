## 模块分级速查（普通版 / 专业版（Pro））

| 场景 | 代表能力 | 普通版覆盖 | PRO增强 |
| :-- | :-- | :-- | :-- |
| 界面与显示 | HUD、动画、样式、可视化 | 高 | 中-高 |
| 系统与进程工具 | 进程/窗口/服务控制 | 中 | 高 |
| 自动化与编排 | 触发器、脚本、宏 | 中 | 高 |
| 网络相关 | B站、歌词、在线检索 | 中 | 中-高 |

## A. 界面与显示

- ClickGUI
- Arraylist / Island / ChatBox 等 HUD 组件
- HUD 编辑器
- 搜索界面，支持搜索模块和配置项
- 大量动画、阴影、样式与字体配置

适用场景：

- 个性化面板
- 运行状态可视化
- 通知与信息聚合

## B. 系统与进程工具

- 进程、窗口、服务枚举与控制
- 音量管理、设备管理、时间管理
- 文件监视、文件管理与恢复

适用场景：

- 本机运维与排查
- 系统行为自动化
- 工具化运作

## C. 自动化与操作编排

- 模块联动
- 快捷键触发
- 命令脚本（ `.jbat` / 宏）
- 定时、条件触发

适用场景：

- 重复操作自动化
- 高效流程串联

## D. 网络相关

- Bilibili：用户、视频、评论、稿件、直播相关命令
- 网易云：歌曲、歌词、覆盖层/歌词增强相关流程
- 文本数据库与在线检索
- 聊天室
- 多种 AI 调用
- 翻译器
- IP 信息
- TCP 连接监视

适用场景：

- 内容创作辅助
- 在线信息聚合

## E. 专业版（Pro）能力

客户端存在一些 专业版（Pro） 专属命令与模块扩展，例如：

- 更完整的在线接口能力
- 扩展 GUI 风格 `Neverlose` 与动画主题
- 更复杂的自动化/筛选逻辑
